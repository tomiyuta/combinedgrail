## OG防御型の位置づけ（FB確定定義）

> **OG防御型は「αのtail riskを薄める低相関バッファ」**
>
> - α↔OG相関: **-0.126（弱逆相関）**
> - 両者が単独でも正の期待値を持つため、ただのヘッジではなく
>   **正の期待値どうしのブレンド** → Sharpe 1.854の源泉
> - OG防御型がDDを削り・Sharpeを上げ・GFC擬似でも部分ヘッジとして機能

## 実務設計（確定版）

### ポジションサイズ


### ポートフォリオ構成
```
α-Grail: 40%（α40% + OG60%の固定比率）
現金:    60%
→ 実効エクスポージャー: α16% + OG24% + 現金60%
```

### 口座配分
```
NISA成長枠 → α成分（高回転・高期待値）優先
その他     → OG成分（低回転）
```

### 追加最適化の禁止
**40:60は固定。比率の再探索・再最適化は行わない。**
将来運用では「先験固定ルール」として扱うことで、実質N=1（PSR=1.000）と読み替えられる。
ただし「過去に11本から選んだ事実」は記録上消えない点に注意。
